 🚀 EduVision — AI-Powered Student Performance Analysis System

## 📌 Overview
EduVision is a Django-based machine learning web application designed to analyze and predict student academic performance using educational and behavioral data.

The project combines:
- Machine Learning
- Data Analysis
- Django Web Framework
- Interactive Dashboards

to help teachers and students gain valuable academic insights.

---

## 🎯 Problem Statement
Educational institutions often struggle to:
- Identify weak students early
- Analyze performance trends effectively
- Provide personalized academic guidance

EduVision solves this problem using predictive analytics and machine learning models.

---

## 💡 Key Features

### 🔮 Performance Prediction
- Predicts student performance category:
  - Excellent
  - Average
  - Poor
- Based on:
  - Attendance
  - Marks
  - Assignments

### 📊 Data Analysis & Visualization
- Confusion Matrix
- Feature Correlation Analysis
- Feature Importance
- Student Performance Distribution

### ⚖️ Machine Learning Models
Models used:
- Logistic Regression
- Decision Tree
- Random Forest

Final selected model:
- ✅ Random Forest

### 👨‍🏫 Teacher Dashboard
- Monitor student performance
- Identify weak students
- View prediction reports
- Analyze academic trends

### 🎓 Student Dashboard
- Subject-wise performance
- Feedback and suggestions
- Performance tracking

### 🤖 Smart Assistance
Provides insights and recommendations for improving academic performance.

### 4️⃣ Model Training
Machine learning models were trained and evaluated.
